# SyobonAction_PS3

Open Syobon Action PlayStation 3 port
Original source code https://sourceforge.net/projects/opensyobon/

# Build

Requires PSL1GHT to build it.
https://github.com/ps3dev/PSL1GHT

# Gameplay

In main menu:  
start - start game,  
up - random level,  
before the start  
select - stage 1,  
square - 2,  
cross - 3,  
circle - 4,  
triangle - 5,  
R1 - 6,  
L1 - 7,  
R2 - 8,  
L2 - 9;  

in game:  
left and right - move,  
Х - jump,  
R3 - exit game,  
L3 - exit to main menu,  
О - suicide.
